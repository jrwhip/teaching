# Teaching

K-6 educational application starting with math, designed to expand into other subjects.

## Getting Started

```bash
pnpm install    # Install dependencies
pnpm start      # Dev server on http://localhost:4200
pnpm build      # Production build to dist/word-list/browser
pnpm lint       # Run ESLint
pnpm test       # Run Vitest
```

## Stack

Angular 21, TypeScript 5.9, Vitest, ESLint 9, SCSS, ag-Grid 35, Font Awesome 6, AWS Amplify (Cognito + AppSync + DynamoDB).
