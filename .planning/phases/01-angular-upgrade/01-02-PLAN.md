<objective>
Remove zone.js and enable zoneless change detection.

Purpose: The app already uses signals throughout. zone.js is dead weight — it patches every async API for change detection the app doesn't need. Removing it improves performance and reduces bundle size.
Output: App runs fully zoneless with provideZonelessChangeDetection().
</objective>

<task type="auto">
  <name>Task 1: Enable zoneless change detection</name>
  <files>jims-app/src/app/app.config.ts, package.json</files>
  <action>
    1. In app.config.ts, add `provideZonelessChangeDetection()` to the providers array (import from @angular/core).
    2. If `provideZoneChangeDetection()` exists, remove it.
    3. Remove zone.js from package.json dependencies.
    4. Remove zone.js import from polyfills (check angular.json polyfills array and any polyfills.ts file).
    5. Run `pnpm install` to update lockfile.

    Do NOT add provideExperimentalZonelessChangeDetection — that was the old API. In Angular 21, provideZonelessChangeDetection() is the stable API.
  </action>
  <verify>`pnpm build` succeeds. `grep -r "zone.js" package.json` returns nothing.</verify>
  <done>zone.js fully removed from dependencies and polyfills. provideZonelessChangeDetection() in app.config.ts. Build passes.</done>
</task>

<task type="auto">
  <name>Task 2: Audit and fix zone-dependent patterns</name>
  <files>jims-app/src/**/*.ts</files>
  <action>
    Search for patterns that depend on zone.js for change detection:

    1. `setTimeout` / `setInterval` used to trigger change detection — replace with signal updates or explicit ChangeDetectorRef.markForCheck() if needed
    2. `ChangeDetectorRef.detectChanges()` — evaluate if still needed, keep if it is
    3. Direct `addEventListener` — ensure any DOM event handlers that update state go through signals
    4. `NgZone.run()` or `NgZone.runOutsideAngular()` — remove, these are meaningless without zone.js
    5. Any `zone` imports — remove

    The app is already signal-based, so most of this should be clean. Fix only what's broken.
  </action>
  <verify>`pnpm build` succeeds. `grep -rn "NgZone\|zone.js" jims-app/src/` returns nothing relevant.</verify>
  <done>No zone-dependent patterns remain. All state changes go through signals. Build passes.</done>
</task>
